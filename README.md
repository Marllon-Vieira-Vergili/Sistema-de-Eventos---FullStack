# Sistema de Eventos - FullStack

Sistema completo para gerenciamento de eventos com frontend em React e backend em Spring Boot.

## Tecnologias utilizadas

**Frontend:**
- React
- Axios
- Bootstrap

**Backend:**
- Java versão 21
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL

## Como rodar

**Backend:**
```bash
cd backend
mvn spring-boot:run

Ou executar pelo Intellij
```
**Frontend:**

cd frontend
npm install vite@latest
npm run dev

Acesse:

Frontend: http://localhost:3000
Backend: http://localhost:5173
Projeto desenvolvido para estudos e prática de desenvolvimento fullstack.

Author: Marllon Vieira Vergili

```
